# Advent of Code 2025

This was my first AOC, where I finished all of the exercises. The hardest one for me was day 10 part
2, for which I had to look up other people solutions and in the end decided on the approach I wanted
to implement.

The challenges from AOC2025 took me a while to fully complete, but it was fun and I learned a lot. I
had a self imposed challenge to:

- use only the C programming language
- use no external libraries

I tried to clean up the code of some days, but the main objective was to solve the challenges, so
the code might be a bit hacky for some days, since I allowed myself to cut some corners on code
quality, if it meant reaching the solutions faster. All the code was handwritten, no LLMs were used
to generate code. I built my `Makefile` with the help of AI.

This was fun!

## Running the solutions

To run the solutions, first clone the repo and move to the cloned folder:

```
git clone https://github.com/Lonec-L/advent-of-code-2025.git && cd advent-of-code-2025
```

Then, add a text file named `in` and place your input into it. Run with `make <day number>`. For
example, for day 10 we would run:

```
make 10
```

> [!NOTE]  
> Tested on Mac OS with Apple clang version 21.0.0
